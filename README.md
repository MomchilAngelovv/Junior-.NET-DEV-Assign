Задача – Junior .Net Developer

1.Да се направи базата от данни на MS SQL.
Нека да има следните таблици.
1.1.Users
Там да се пазят името и паролата на потребителите, които ще влизат в системата.
1.2.Contragents
Там да се пазят клиентите, които имаме. За тях ни трябват:Име,Адрес,Мейл и ДДС номер.Нека всеки клиент да е вързан към даден потребител.Един клиент може да е свързан само към един потребител.
2.Да се направи .NET WEB API проект, който да служи за връзка с базата от данни.
Там ще ти трябват методи за:
2.1.Създаване на потребители
2.2.Редакция на потребители
2.2.Взимане на данни за даден потребител по Ид
3.Да се направи .NET MVC проект, който да служи за фроент-енд.
3.1. Да се направи страница за регистрация на потребители.
3.2.Да се направи страница за логин.
3.3.Да се преработи логването да работи с нашите потребители(да се пренапише MembershipProvider).
3.4.Да се направи страница, за добавяне на клиенти в базата.
3.5.Да се направи страница за показване на всички клиенти за даденият потребител. Може да е в табличен вид.
3.6.След натискане върху ред с клиент, да се отварят детайли за даденият клиент.
Там ще показваме всичките данни на клиента:Име,Адрес,Мейл и ДДС номер.
